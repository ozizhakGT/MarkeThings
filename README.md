# MarkeThings

###GETTING STARTED

## Open Terminal/CMD into the Root folder path
## Write Commands for install node_modules and run webpack live server automaticlly
```
npm install
npm start 
```
 ## If you want to build for production 
 ```
 npm  run build
 ```
 ## If you want to build for Development
 ```
 npm run dev
 ```
